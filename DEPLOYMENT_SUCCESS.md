## 🌐 自定义域名配置

如果您想使用自定义域名，请：

### 1. 运行域名验证脚本
```bash
./validate-domain.sh
```

### 2. 配置 DNS 记录
在您的域名提供商处添加：
```
类型: A
名称: @
值: 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153

类型: CNAME
名称: www
值: Khangey.github.io
```

### 3. 设置 GitHub Pages
- 在 "Custom domain" 字段输入您的域名
- 勾选 "Enforce HTTPS"
- 点击 "Save"
